# Firebase App – Child Weight Classification

# Overview



\# Child Weight Classification App



An Android application that integrates a Feedforward Neural Network (FNN) model to classify children's weight status based on height, weight, and age.



The app allows users to input a child’s:



* height
* weight
* age in months
* gender



Using these inputs, the integrated Feedforward Neural Network (FNN) model predicts whether the child falls under one of the following categories:



* normal
* overweight
* obese
* underweight
* severely underweight



#### Key Features



Simple and user-friendly input interface

Real-time classification using a trained ML model

Integration of machine learning within a mobile application

Lightweight and focused on essential health indicators

Machine Learning Integration



The application uses a pre-trained Feedforward Neural Network (FNN) model. The model was trained on structured child health data to identify patterns.



The goal of the model is to approximate classification outcomes based on learned patterns from real-world data.



#### Data \& Privacy Notice



The dataset used to train the model is not included in this repository due to privacy and sensitivity considerations.



Training data was sourced from a specific population group

Only anonymized attributes such as height, weight, and age were used for modeling

No personally identifiable information is included or stored in this project

Purpose



This project was developed as a practical implementation to explore the integration of a trained machine learning model into an Android application environment.



It demonstrates how predictive models can be embedded into mobile apps to provide simple, data-driven insights.
